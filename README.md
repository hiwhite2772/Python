# Python
Dưới đây là một bản **học Python từ cơ bản → nâng cao** được rõ ràng, có lộ trình và tài nguyên để bạn tự học hiệu quả:

# 🐍 Python Learning Roadmap (Beginner → Advanced)

## 📌 1. Giới thiệu Python

Python là một ngôn ngữ lập trình:

* Dễ đọc, dễ học (syntax gần với tiếng Anh)
* Đa mục đích: web, AI, data, automation, game...
* Có cộng đồng rất lớn

👉 Ứng dụng phổ biến:

* Web: Django, Flask
* Data Science: Pandas, NumPy
* AI/ML: TensorFlow, PyTorch
* Automation / scripting

---

## 🚀 2. Giai đoạn 1: Cơ bản (Beginner)

🎯 Mục tiêu: Hiểu cách Python hoạt động và viết được chương trình đơn giản

📚 Nội dung cần học:

* Cài đặt Python & IDE (VS Code / PyCharm)
* Cú pháp cơ bản:
  - Biến (int, float, str, bool)
  - Input / Output
  - Toán tử
* Điều kiện: if, elif, else
* Vòng lặp: for, while
* Cấu trúc dữ liệu: List, Tuple, Set, Dictionary
* Hàm: def, tham số, return

🧠 Ví dụ: python
```
def greet(name):
    return f"Hello {name}"

print(greet("White"))
```

💡 Bài tập:
* Viết máy tính đơn giản
* Kiểm tra số nguyên tố
* Game đoán số

---

## ⚙️ 3. Giai đoạn 2: Trung cấp (Intermediate)

🎯 Mục tiêu: Hiểu sâu hơn về cách tổ chức code và làm việc thực tế

📚 Nội dung:

* OOP (Lập trình hướng đối tượng)
 - Class, Object
 - Inheritance, Encapsulation, Polymorphism
* Modules & Packages
* File handling: Đọc/ghi file (.txt, .json)
* Exception handling: try, except
* List comprehension
* Lambda function
* Virtual Environment (venv)

🧠 Ví dụ: python
```
class Dog:
    def __init__(self, name):
        self.name = name
    def bark(self):
        print(f"{self.name} says woof!")
```

💡 Project:

* To-do list app
* File organizer
* Web scraping đơn giản

---

## 🔥 4. Giai đoạn 3: Nâng cao (Advanced)

🎯 Mục tiêu: Làm project thực tế và hiểu sâu hệ sinh thái Python

📚 Nội dung:

* Decorators
* Generators (yield)
* Multithreading / Multiprocessing
* Async / Await
* Design patterns cơ bản
* Testing (pytest)
* Logging

## 📅 7. Lộ trình đề xuất (8–12 tuần)

| Tuần | Nội dung               |
| ---- | ---------------------- |
| 1–2  | Cú pháp cơ bản         |
| 3–4  | Hàm + cấu trúc dữ liệu |
| 5–6  | OOP + file             |

| 9–12 | Chọn hướng chuyên sâu  |
